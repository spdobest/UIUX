one-value syntax: the single value sets the color of all four border sides.

two-value syntax: the first value sets the color of the top and bottom border, while the second value sets the color of the right and left sides border.

three-value syntax: the first value sets the color of the top border, the second value sets the color of the right and left border, and the third value sets the color of the the bottom border.

four-value syntax: each value sets the color of the border individually in the order top, right, bottom, and left.
